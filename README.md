# Personal-intro-python
python-personal introduction program

Project Overview:
This project is a simple Python program that introduces me by displaying my name, age, and hobbies. It demonstrates basic Python concepts such as variables, lists, and print statements.

Setup Instructions:
1. Install Python on your system.
2. Download the project files.
3. Open a terminal in the project folder.
4. Run the following command:
python personal_intro.py

Code Structure:
- personal_intro.py – Main Python program
- requirements.txt – Lists project dependencies
- screenshot.png – Screenshot of program output
- README.md – Project documentation

Visual Documentation:
The file screenshot.png contains the output generated by the Python program.


This project uses:
- Variables
- String formatting
- Print statements

Testing Evidence:
The program was executed successfully, and the output screenshot is included in the repository.


Through this project, I learned:

- How to create and use variables in Python
- How to store data in lists
- How to display information using print statements
- How to organize a project using GitHub
